# Huanzi's Beauty Sink — 项目说明文档

## 一、产品定位

**一句话简介**

> 我们做了一款面向**对美妆洗护感兴趣的年轻用户**，在**碎片化娱乐时间**中，通过**把 350+ 品类护肤彩妆产品「按潜在规律排列」的轻量化排列互动机制**，带来**洗护美妆产品的认知满足感与收纳掌控感**的产物——同时以产品图鉴收集、自由摆放、使用流程关卡为载体，天然契合电商种草与品类教育场景，具备极大的品牌联名与导购商业化潜力。

**核心价值主张**

| 维度     | 描述                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 用户价值 | 在游戏中建立美妆洗护产品认知，了解品类规律与使用顺序                         |
| 情绪价值 | 「整理台面」的掌控感、收集图鉴的成就感、轻松闯关的愉悦感                     |
| 商业价值 | 产品曝光自然、品类教育高效、种草链路短——用户在认识产品的同时即完成兴趣转化 |

---

## 二、目标用户

- **主要用户**：18-35 岁，对美妆护肤有兴趣但认知尚浅的年轻女性
- **使用场景**：通勤、休息、等待等碎片化时间（单局 1-3 分钟）
- **用户痛点**：买了很多产品但不清楚它们叫什么、怎么分类、如何按顺序使用

---

## 三、核心玩法机制

游戏以「梳妆台整理」为世界观，玩家扮演整理师，将散落在洗手盆里的美妆洗护产品归位。

### 3.1 三大题型

#### 🔢 排序题（Sort）

**规则：** 提示区展示3件已排好的物品（高矮/宽窄顺序），玩家从池子里找出3件继续完成序列。

**排序维度：**

- 高度从低到高 / 从高到低（按真实产品高度 cm 分档：迷你 <4cm、小号 4-9cm、中号 9-15cm、大号 15-20cm、超大号 20-24cm、巨大号 ≥24cm）
- 宽度从窄到宽 / 从宽到窄（按真实宽度分档：<2cm / 2-4cm / 4-6cm / 6-8cm / 8-10cm / ≥10cm）

**设计亮点：** 每档优先选「高度感最明显」的瓶管类物品，避免扁盒、球形等视觉干扰物，确保玩家一眼能感知高矮/宽窄差异。

---

#### 🌿 流程题（Process）

**规则：** 左上角出现半透明场景提示（如「晚上回家后。先卸妆清洁」），提示区展示3个步骤示例，玩家从含3件干扰项的池子里找出正确的后续3步完成使用流程。

**流程主题（共10条）：**

| 主题           | 场景           | 流程顺序                                               |
| -------------- | -------------- | ------------------------------------------------------ |
| 早间护肤       | 早上出门前     | 清洁→水类→精华→保湿→保湿→防晒                     |
| 夜间护肤       | 晚上回家后     | 卸妆清洁→清洁→水类→精华→眼部护理→保湿             |
| 洗护护理       | 洗完头洗完澡   | 头发清洁→护理→加强护理→身体清洁→身体滋润→局部护理 |
| 彩妆整理       | 完成妆容       | 底妆→遮瑕→定妆→面部彩妆→睫毛产品→唇妆             |
| 护肤身体全流程 | 洗澡后完整护理 | 身体清洁→去角质→身体滋润→局部护理→香氛→家居香氛   |
| 完整上妆       | 护肤到彩妆     | 保湿→防晒→底妆→遮瑕→眼部护理→唇妆                 |
| 深层清洁       | 周末护理日     | 身体去角质→身体清洁→滋润→局部护理→香氛→家居香氛   |
| 眼部精细护理   | 细致眼部护理   | 卸妆清洁→清洁→水类→眼部护理→保湿→防晒             |
| 早间出门全流程 | 完整早间准备   | 清洁→水类→保湿→防晒→底妆→遮瑕                     |
| 头皮深层护理   | 头发深层滋养   | 头发清洁→护发→加强护理→身体清洁→去角质→局部护理   |

**设计亮点：** 题目选用美妆行业公认的最通用使用顺序，具备品类教育价值。干扰项3件，不会让题目过难，但足以训练用户辨别产品属性。

---

#### 🔍 排除题（Exclude）

**规则：** 提示区展示3件符合同一特征的物品（如全部粉紫色），玩家从一个大池子（15件混排）里找出3件具有相同特征的物品放入槽位。

**判断逻辑：** 不要求顺序，只要3件都满足条件即过关。

**排除主题（共10条）：**

| 关卡   | 题目             | 筛选条件                               |
| ------ | ---------------- | -------------------------------------- |
| 第3关  | 找出粉紫色物品   | colorFamily 为粉红或紫色               |
| 第6关  | 找出圆罐         | shapeType 为圆罐（排除眼霜等边界物品） |
| 第9关  | 找出头发护理产品 | category 为头发护理                    |
| 第12关 | 找出身体护理产品 | category 为身体护理                    |
| 第15关 | 找出小号物品     | 真实高度 <9cm 且宽度 <8cm              |
| 第18关 | 找出大瓶物品     | 大号/超大号/巨大号档位                 |
| 第21关 | 找出扁盒         | shapeType 为扁盒，宽>高×1.5           |
| 第24关 | 找出护肤品       | bigCategory 为护肤                     |
| 第27关 | 找出彩妆品       | bigCategory 为美妆                     |
| 第30关 | 找出泵瓶         | shapeType 为泵瓶，高度 >15cm           |

**设计亮点：** 提示区样本经严格筛选取最典型物品，避免歧义边界物。池子混入跨品类干扰，训练产品识别能力。

### 3.2 游戏模式

| 模式               | 描述                                                              |
| ------------------ | ----------------------------------------------------------------- |
| **闯关模式** | 共30关，按「排序→流程→排除」三关一循环结构推进，难度梯度设计    |
| **无限模式** | 计时挑战（1/3/5分钟可选），从排序题和流程题中随机出题，冲击排行榜 |
| **自由模式** | 选择场景和物品自由摆放，支持拍照保存到回忆相册                    |
| **PK模式**   | 房间码联机对战，比拼速度（开发中）                                |

## 四、玩法体验流程

```
进入游戏
  ↓
创建玩家档案（选头像 + 昵称）
  ↓
选择模式
  ├─ 闯关模式：选关卡 → 开始
  ├─ 无限模式：选时长 → 开始
  └─ 自由模式：选场景 → 选物品 → 自由摆放

游戏中（以闯关为例）
  ↓
HUD 显示关卡/计时 + 左上角题型提示（流程/排除题才显示）
  ↓
序列区：展示3件示例物品（已排好）+ 3个待填槽位
  ↓
洗手盆：散落待选物品（排序题3件/流程题6件/排除题15件）
  ↓
玩家点击/拖拽选中物品 → 放入槽位
  ↓
点击「检查」
  ├─ 全对 → 过关弹窗 + 本关用到的产品收录图鉴 → 下一关
  └─ 有错 → 标红提示 + 可继续修改（提示道具最多用3次）
  ↓
30关全通 → 通关总结
```

## 五、产品图鉴系统

- **收录机制**：每完成一关（任意模式），本关涉及的产品即时收录入图鉴，无需等级解锁
- **图鉴内容**：按品类分组（护肤、美妆、洗护、香氛、工具），未收录产品显示「???」问号
- **统计展示**：已收录 X / 总计 Y 种产品

**商业化潜力**：图鉴卡片可嵌入真实产品链接、成分介绍、使用教程——玩家认识产品的瞬间即是种草转化的最佳时机。

## 七、技术架构

- **前端**：纯原生 HTML + CSS + JavaScript，无框架依赖
- **数据存储**：浏览器 localStorage（玩家档案、闯关进度、图鉴收录、回忆相册）
- **资产**：543件产品的AI生成插画，统一美术风格
- **跨端**：响应式设计，支持手机横竖屏与桌面浏览器
